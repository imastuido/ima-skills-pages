技能列表

IMA XSH Pro

小红书种草转化技能。EN: Xiaohongshu seeding conversion ops with review-ready delivery packs.

技能亮点 / Why Teams Choose It

  • 支持三种内容模式：图文笔记、短视频、双模式并行。
  • 从策略到执行一次交付：文案、封面提示、执行计划、发布审核包。
  • 非常适合“品牌种草 + 转化跟进”一体化运营流程。

推荐输入 / Recommended Inputs

  1. 品牌、产品、核心卖点。
  2. 目标（awareness / engagement / conversion）。
  3. 内容模式（image_post / short_video / both）。
  4. 受众画像、预算区间、发布时间窗口。

标准产出 / Standard Deliverables

  • 主文案包（xhs_master_copy）。
  • 封面与视觉提示包（cover prompt pack）。
  • 视频脚本包（短视频模式）。
  • 生成执行包 + 风险与审核包 + 发布包。

这些产物可以直接进入“内部评审 -> 执行制作 -> 发布排期”流程。

边界说明 / Scope

该技能专注于“内容与执行交付”，不替代平台账号的最终人工发布决策。
